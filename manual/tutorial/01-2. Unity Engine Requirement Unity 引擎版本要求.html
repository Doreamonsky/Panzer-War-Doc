    01-2. Unity Engine Requirement Unity 引擎版本要求

    The following softwares are required:

    1. Unity 2021.3.29f1 (Must be the same version)

    You can install Unity from this link: 2021.3.29f1 To install unityhub, use this link https://public-cdn.cloud.unity3d.com/hub/prod/UnityHubSetup.exe If you have unity hub installed, you can install by this link: **unityhub://2021.3.29f1/204d6dc9ae1c **

    1. Unity 2021.3.29f1 （需要相同版本）

    你可以从这里安装 Unity: 2021.3.29f1 安装 unity hub，请使用此连接 https://unity.cn/releases 如果你已经安装了 Unity Hub 则可以在游览器中输入以下连接安装 **unityhub://2021.3.29f1/204d6dc9ae1c **
