    25. Runtime Editor Debugger 编辑器实时调试功能

    To simplify mod debugging, you can directly run mods in unity editor. 调试 Mod 一直是一件繁琐的事情，所以为了简化 Mod 开发流程，增加了编辑器内直接加载 Mod 的功能。

    Usage 使用方法

    Package Mods 打包模组

    Click Build Linked Objects to AssetBundle 点击 编译相关资源 打包你的载具模组

    Open Scene and Run 打开场景并运行

    场景位于： Packages/com.shanghaiwindy.middlelayer/Tools/RuntimeDebugger/VehiclePlayground.unity 如图所示 选择你需要调试的载具进行使用
