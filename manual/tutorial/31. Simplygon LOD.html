    Simplygon LOD

    Simplygon 是由微软开发的模型减面工具。

    Install the sdk 安装 SDK

    You need to install the sdk provided from simplygon. And choose free license (with 200 prorcess limit per day). Download the sdk from https://www.simplygon.com/downloads

    你需要安装由 Simplygon 提供的 sdk，并选择免费证书进行激活 （免费证书每天可处理 200次）你可以从官网下载 sdk https://www.simplygon.com/downloads

    Install the package to editor 将 Package 装到 Editor 上

    Click on + and select Add package from disk ... to add this local package from Panzer War Mod folder 点击 + 选择 Add package from disk ... 从装甲纷争的模组文件夹中选择这个 json 文件

    Toggle on simplygon support in editor 启用 Simplygon 支持

    Then repack you vehicles. Your vehicles will use lods generated from Simplygon.

    然后点击载具上的 Repack，你的载具会使用由 Simplygon 提供的 Lod 模型
